
* ls  				=  lista directorios y archivos
* pwd  				=  en que directorio estamos
* git init				=  inicializa un proyecto nuevo
* git status			=  ver el estado de nuestros archivos
* git add				= agrega un archivo al Staging area
* git config --global user.email "..."	= configura el Email del usuario
* git config --global user.name "..."	= configura el nombre del usuario
* git log				= para ver todos los commits que hemos creado
* git checkuot --			= revierte cambios en los archivos
* git diff				= muestra las diferencias hechas en los archivos
